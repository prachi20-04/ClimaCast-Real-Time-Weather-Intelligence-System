# ClimaCast-Real-Time-Weather-Intelligence-System
ClimaCast is a smart weather monitoring tool that fetches live data, predicts future conditions using machine learning, and displays insights through a user-friendly dashboard. It supports location-based tracking, alerts for extreme weather, and helps users in agriculture, travel, and disaster planning.

Real-Time Weather Monitoring
Fetches live weather data using APIs based on user location or input.
Predictive Analytics
Uses machine learning models to forecast temperature, humidity, and rainfall trends.
Interactive GUI Dashboard
Displays weather metrics and predictions in a user-friendly interface using Tkinter.
Location-Based Alerts
Warns users about extreme weather conditions for proactive decision-making.
Modular Architecture
Separates data fetching, prediction, visualization, and user interaction for maintainability.
